## Engineering Principles

![Event Driven](https://img.shields.io/badge/Event--Driven%20First-231F20?style=flat-square&logo=apachekafka&logoColor=white)
&nbsp;Full service decoupling via message brokers. State as immutable events.

![Observability](https://img.shields.io/badge/Observability%20by%20Design-0052CC?style=flat-square&logo=datadog&logoColor=white)
&nbsp;Distributed tracing via correlation IDs. Every action reconstructible from the log.

![Side Effect Isolation](https://img.shields.io/badge/Side--Effect%20Isolation-2E7D32?style=flat-square&logo=checkmarx&logoColor=white)
&nbsp;Dry-run before deploy. No live consequences during debugging.

![CS Fundamentals](https://img.shields.io/badge/Deep%20CS%20Fundamentals-6C3483?style=flat-square&logo=academia&logoColor=white)
&nbsp;Understand WHY things work, not just HOW to use them.
